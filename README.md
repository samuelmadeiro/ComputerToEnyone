Aqui está meus registros do meu software de assessibilidade, buscando alcançar pessoas que são impossibilitadas de mexerem o cursor do computador. Usando apenas os olhos e a boca os usuários podem navegar e fazer atividades cotidianas, como usar o youtube ou o instagram, promovendo assessibilidade e inclusão para as pessoas com essa dificuldade. A cada versão tentarei melhorar a precisão do cursor e as opções do meu software.
